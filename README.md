# Ecommerce Amiibos

Implementación con react-native, react-navigation, redux.

  - Carrito de Compras
  - Lista de Productos


### Installation

Requires [Node.js]
Requires [Python]
Requires [JDK8]
Requires [ADB]


```sh
$ git clone https://github.com/jeremyroa/amiibos.git
$ cd amiibos
$ yarn install
$ yarn start
// Otra ventana (Si tienes el Android emulator o ya en Debugging el teléfono)
$ open device/android
```
